<h2>Features</h2>

<CardGrid>
  <Card title="Simple" icon="rocket">
    Publish all your car listings in one simple JSON file that can easily be read by any data consumer.
  </Card>
  <Card title="Complete" icon="document">
    The format supports many car features, as well, as multiple showrooms, and many other dealer details.
  </Card>
  <Card title="Standard" icon="open-book">
    The format is based on the widely-used Data Package standard, making it compatible with its software ecosystem.
  </Card>
  <Card title="Easy-to-use" icon="laptop">
    The format comes with Python and TypeScript SDKs, simplifying the process of publishing and consuming data.
  </Card>
</CardGrid>
