## Features

<CardGrid>
  <Card title="Lightweight" icon="rocket">
    Fairspec defines a few simple JSON-based metadata formats for describing catalogs, datasets, tables, and files.
  </Card>
  <Card title="Compatible" icon="open-book">
    Fairspec is compatible to the widely-used, de-facto standard, Datacite and JSON Schema specifications.
  </Card>
  <Card title="Flexible" icon="document">
    Fairspec is flexible enough to describe any kind of data, including tabular data, images, and geospatial data.
  </Card>
  <Card title="Software" icon="laptop">
    Fairspec is software-first, it comes with Python and TypeScript implementations out-of-the-box.
  </Card>
</CardGrid>
